01

主語とsein動詞

英語の be動詞（am / are / is）にあたるもの

まずは文の土台、「〜です／〜だ」を表す sein（ザイン）動詞から。英語の be動詞と同じ役割ですが、主語によって形がガラッと変わります。

主語（人称代名詞）× sein 一覧

主語ごとに sein の形が変化する
主語（ドイツ語）意味sein の形
ich イッヒ私はbin ビン
du ドゥー君はbist ビスト
er / sie / es エア／ズィー／エス彼／彼女／それはist イスト
wir ヴィア私たちはsind ズィント
ihr イーア君たちはseid ザイト
sie / Sie ズィー彼ら／あなた（丁寧）はsind ズィント

英語 ⇄ ドイツ語 くらべてみる

English

形は am / are / is の 3種類だけ。you も we も they もぜんぶ are で済む。

Deutsch

形は bin / bist / ist / sind / seid の 5種類。主語ごとに覚える必要がある。

💡 ここが英語と決定的に違う：英語より変化が激しい。特に「私＝bin」「君＝bist」は英語にない専用の形。

🤝 ここは英語と似ている：「主語＋be動詞＋◯◯」で「〜です」を作る語順は同じ。

例文で確認

Ich bin Student.イッヒ ビン シュトゥデント私は学生です。（英：I am a student.）
Du bist nett.ドゥー ビスト ネット君は親切だ。（英：You are nice.）
Er ist müde.エア イスト ミューデ彼は疲れている。（英：He is tired.）

02

一般動詞の現在形と人称変化

lernen（学ぶ）, machen（する）などの規則動詞

次は「学ぶ」「作る」といった普通の動詞。ドイツ語では、語幹（変わらない部分）に主語ごとの語尾をくっつけて使います。

lernen（レルネン＝学ぶ）の変化

語幹は lern-。ここに語尾 -e / -st / -t / -en を付ける。

主語形語尾
ichlerne レルネ-e
dulernst レルンスト-st
er / sie / eslernt レルント-t
wirlernen レルネン-en
ihrlernt レルント-t
sie / Sielernen レルネン-en

英語 ⇄ ドイツ語 くらべてみる

English

変わるのは 三人称単数のときだけ。He / She のときだけ動詞に -s（learns）。あとは全部 learn のまま。

Deutsch

すべての主語で語尾が変わる。ich / du / er / wir…それぞれ専用の語尾がある。

💡 ここが英語と決定的に違う：英語は「He/She のときだけ -s を足す」だけ覚えればOK。ドイツ語は6パターン全部が変化するので、語尾のセットを丸ごと覚える。

🤝 ここは英語と似ている：「主語によって動詞が少し変わる」という発想そのものは英語（-s）と同じ。ドイツ語はそれを全主語に拡張した版。

例文で確認

Ich lerne Deutsch.イッヒ レルネ ドイチュ私はドイツ語を学ぶ。（英：I learn German.）
Du machst Sport.ドゥー マッハスト シュポルト君はスポーツをする。（英：You do sports.）
Sie lernt Englisch.ズィー レルント エングリッシュ彼女は英語を学ぶ。（英：She learns English.）

03

名詞の「性別」と「〜を（4格）」の壁

der / die / das と、目的語での変化

ドイツ語最初の大きな壁。すべての名詞に男性・女性・中性という「性別」があり、「the」にあたる冠詞がそれぞれ違います。さらに「〜を」（4格＝目的語）になると形が変わります。

定冠詞（the）は性別で3種類

性別「〜は」1格「〜を」4格
男性 (m)der Mann デア マン（男）den Mann デン マン
女性 (f)die Frau ディー フラウ（女）die Frau ディー フラウ
中性 (n)das Kind ダス キント（子）das Kind ダス キント

ポイント：4格で形が変わるのは男性だけ！ der → den に化ける。女性 die・中性 das はそのまま。まずはこの一点を押さえればOK。

英語 ⇄ ドイツ語 くらべてみる

English

「the」は 1種類だけ。the man も the woman も the child も、主語でも目的語でも ずっと the。

Deutsch

「the」が 性別と格で変化。der / die / das があり、目的語では der が den に変わる。

💡 ここが英語と決定的に違う：英語は冠詞を一切気にしなくていい。ドイツ語は「この名詞は何性か」＋「今どの格か」をいつも意識する必要がある。

例文で確認

Ich sehe den Mann.イッヒ ゼーエ デン マン私はその男性を見る。（der→den に変化！）
Ich habe einen Hund.イッヒ ハーベ アイネン フント私は犬を飼っている。（ein も男性4格で einen に）
Ich lese das Buch.イッヒ レーゼ ダス ブーフ私はその本を読む。（中性なので das のまま）

04

否定文と疑問文の作り方

動詞をひっくり返す・nicht と kein の使い分け

ここは英語よりむしろ簡単に感じる人も多いところ。疑問文は「動詞と主語をひっくり返すだけ」、否定は nicht か kein を置くだけです。

① 疑問文：動詞と主語を入れ替えるだけ

Du lernst Deutsch.君はドイツ語を学ぶ。（平叙文）
↓ 動詞を先頭へ
Lernst du Deutsch?レルンスト ドゥー ドイチュ？君はドイツ語を学んでるの？

② 否定：nicht と kein の使い分け

使う語何を否定する？例
nicht ニヒト動詞・形容詞・文全体Ich trinke nicht.
飲まない
kein カイン名詞（a / 無冠詞のもの）Ich habe kein Auto.
車を持っていない

英語 ⇄ ドイツ語 くらべてみる

English

一般動詞だと do / does が必要。Do you learn? / I don't learn. と、助動詞 do を借りてくる。

Deutsch

do にあたる助動詞は不要。動詞そのものを前に出す／nicht を置くだけ。

💡 ここが英語と決定的に違う：英語の「Do you〜?」「don't」のような do の助けを一切使わない。

🤝 ここは英語と似ている：be動詞の疑問文 「Are you〜?」を作るノリ（動詞を前に出す）が、ドイツ語ではすべての動詞で起きる、と考えると腑に落ちる。

例文で確認

Bist du glücklich?ビスト ドゥー グリュックリッヒ？君は幸せ？（英：Are you happy?）
Ich trinke nicht.イッヒ トリンケ ニヒト私は飲まない。（動詞を否定）
Ich habe keine Zeit.イッヒ ハーベ カイネ ツァイト私は時間がない。（名詞を否定＝kein）

05

話法の助動詞と「枠構造」

können / müssen と、動詞が文末に飛ぶルール

最後は語順のパズル。können（〜できる）や müssen（〜しなければ）といった助動詞を使うと、本来の動詞が文の一番うしろに吹っ飛びます。これを「枠構造（Klammer）」と呼びます。

🔲 枠構造を目で見る

Ich kann ← 助動詞は2番目
Deutsch sprechen ← 本動詞は文末（原形）

＝ kann … sprechen の2語で文全体を挟む「枠（カッコ）」を作るイメージ。

英語 ⇄ ドイツ語 くらべてみる

English

I can speak German.
can と speak が となり同士。目的語 German は最後。素直な語順。

Deutsch

Ich kann Deutsch sprechen.
直訳すると「I can German speak」。動詞 sprechen が 文末へ。

💡 ここが英語と決定的に違う：英語では助動詞と動詞がくっつく（can speak）。ドイツ語は助動詞と本動詞が離れて文を挟む。目的語が真ん中に入り込むのが最大の違い。

können（〜できる）の変化も確認

主語形
ichkann カンich と er が同じ形！
dukannst カンスト
er / sie / eskann カン-t が付かない
wir / sie / Siekönnen ケネン
ihrkönnt ケント

例文で確認

Ich muss jetzt gehen.イッヒ ムス イェツト ゲーエン私は今行かなければならない。（gehen が文末）
Kannst du schwimmen?カンスト ドゥー シュヴィンメン？君は泳げる？（疑問文＋文末の schwimmen）
Wir wollen Pizza essen.ヴィア ヴォレン ピッツァ エッセン私たちはピザを食べたい。（essen が文末）
